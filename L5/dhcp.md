# DHCP

如果逐一为每台主机配置IP地址会非常繁琐，为了实现自动化统一管理ip地址分配，就产生了动态主机设置协议（英语：Dynamic Host Configuration Protocol，缩写：DHCP）；它是一个用于局域网的网络协议，位于OSI模型的应用层，使用UDP协议工作，主要有两个用途：

* 用于内部网或网络服务供应商自动分配IP地址给用户
* 用于内部网管理员作为对所有计算机作中央管理的手段
